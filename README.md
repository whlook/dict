命令行下中英文翻译工具（Chinese and English translation tools in the command line），翻译服务基于有道翻译。

[![asciicast](http://asciinema.org/a/123670.png?v=2)](http://asciinema.org/a/123670)


## 安装(Install)

```bash
pip install dict-cli
```
或者本地安装
```bash
python -m pip install -e .
```

## 用法(Usage)

#### 英译中(English To Chinese)
```bash
$ dict test
###################################
#  test 测试
#  (U: tɛst E: test)
#
#  n. 试验；检验
#  vt. 试验；测试
#  vi. 试验；测试
#  n. (Test)人名；(英)特斯特
#
#  Test : 测试
#          测验
#          检验
#  Test Drive : Test Drive
#                Test Drive
#                无限狂飙
#  Test Engineer : 测试员
#                   测试工程师
#                   软件测试工程师
###################################

$ dict I love you
###################################
#  I love you 我爱你
#
#
#  我爱你。
#
#  I love you : 我爱你
#                ILOVEYOU蠕虫
#                寻找伴郎
#  I really love you : 真的爱你
#                       其实很爱你
#                       我是真的爱你
#  I Do love you : 我是爱你的
#                   真的爱你
#                   爱你我该怎么办
###################################
```
